# Flight Dashboard Component

The Flight Dashboard Component is the central hub of the SkyTracker project, providing an interface to view and interact with various flight-related data, including real-time flight movements and airport information.

## Features

- Displays real-time flight movements on a map
- Provides airport information such as arrivals, departures, and gate details
- Integrates with various APIs for up-to-date information
